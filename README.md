# lkx-cli

## Installation

```sh
npm i lkx-cli -g
```

## Usage

```sh
Options:
  -V, --version      output the version number
  -h, --help         display help for command

Commands:
  list               查询可用的项目模板
  create <app-name>  创建一个新项目
  info               打印环境信息
  help [command]     display help for command
```

## 包说明：

- commander // 命令包
- download-git-repo // 下载模版包
- inquirer // 交互包
- ora // 下载提示
